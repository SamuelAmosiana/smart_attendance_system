This directory stores captured face images.

Structure:
  dataset/
  └── faces/
      ├── 2024_BCS_001/       ← student_id (/ replaced with _)
      │   ├── 2024_BCS_001_001.jpg
      │   ├── 2024_BCS_001_002.jpg
      │   └── ...
      ├── 2024_BCS_002/
      │   └── ...
      └── ...
  └── encodings.pkl           ← Auto-generated by encode_faces.py

Instructions:
  1. Use the Admin Dashboard > Register page > Capture Faces button
     (or run:  python face_recognition_module/capture_faces.py --student_id YOUR_ID)
  2. After capturing for all students, click Encode All Faces
     (or run:  python face_recognition_module/encode_faces.py)
  3. Start recognition from the dashboard.
